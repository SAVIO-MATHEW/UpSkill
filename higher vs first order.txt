### **First-Order Component vs Higher-Order Component (React)**

|             | **First-Order Component (FOC)** | **Higher-Order Component (HOC)**                                     |
| ----------- | ------------------------------- | -------------------------------------------------------------------- |
| Meaning     | Normal React component          | Function that takes a component and returns a new enhanced component |
| Purpose     | Renders UI directly             | Reuses logic across components                                       |
| Type        | Component                       | Function â†’ Component                                                 |
| Reusability | Low                             | High                                                                 |

### ðŸš€ Higher-Order Component

Enhances another component.

```jsx
function withLoading(Component) {
  return function EnhancedComponent({ isLoading, ...props }) {
    if (isLoading) return <p>Loading...</p>;
    return <Component {...props} />;
  };
}
```

Usage:

```jsx
const ButtonWithLoading = withLoading(Button);
```
