# ğŸ”¥ What is Semantic HTML?

Semantic HTML means using HTML elements that **clearly describe their meaning and purpose**.

Instead of using generic `<div>` and `<span>`, we use meaningful tags like:

* `<header>`
* `<nav>`
* `<section>`
* `<article>`
* `<footer>`
* `<main>`
