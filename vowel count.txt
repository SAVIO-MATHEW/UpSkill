Counting vowels in a string is straightforward. Here's a simple implementation in JavaScript:

### **Example Code**
```javascript
function countVowels(str) {
  const vowels = "aeiouAEIOU";
  let count = 0;

  for (const char of str) {
    if (vowels.includes(char)) {
      count++;
    }
  }

  return count;
}

// Example usage:
const inputString = "Hello World!";
const vowelCount = countVowels(inputString);
console.log(`Number of vowels: ${vowelCount}`); // Output: "Number of vowels: 3"
```

### **Explanation**
1. Define the string of vowels (`aeiouAEIOU`) to cover both uppercase and lowercase characters.
2. Iterate through each character in the input string.
3. Check if the character exists in the `vowels` string using `includes()`.
4. Increment the counter for every match.

---

### **Alternative Using Regular Expressions**
For a concise approach, you can use a regular expression:
```javascript
function countVowelsUsingRegex(str) {
  const match = str.match(/[aeiouAEIOU]/g);
  return match ? match.length : 0;
}

// Example usage:
const inputString = "Hello World!";
const vowelCount = countVowelsUsingRegex(inputString);
console.log(`Number of vowels: ${vowelCount}`); // Output: "Number of vowels: 3"
```
