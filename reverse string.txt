To reverse a string in JavaScript, you can use a combination of methods to make it simple and efficient. Here's an example:

```javascript
function reverseString(str) {
  return str.split('').reverse().join('');
}

// Example usage:
const originalString = "hello";
const reversedString = reverseString(originalString);
console.log(reversedString); // Output: "olleh"
```

### **Explanation**
1. **`split('')`**: Splits the string into an array of individual characters.
   - `"hello"` becomes `["h", "e", "l", "l", "o"]`.
2. **`reverse()`**: Reverses the order of elements in the array.
   - `["h", "e", "l", "l", "o"]` becomes `["o", "l", "l", "e", "h"]`.
3. **`join('')`**: Joins the array elements back into a single string.
   - `["o", "l", "l", "e", "h"]` becomes `"olleh"`.

Certainly! Apart from the `split-reverse-join` method, here are other ways to reverse a string in JavaScript:

### **1. Using a For Loop**
```javascript
function reverseStringUsingLoop(str) {
  let reversed = '';
  for (let i = str.length - 1; i >= 0; i--) {
    reversed += str[i];
  }
  return reversed;
}

// Example usage:
console.log(reverseStringUsingLoop("hello")); // Output: "olleh"
```
This method iterates through the string from the end to the beginning and appends each character to a new string.

---

### **2. Using Recursion**
```javascript
function reverseStringRecursively(str) {
  if (str === "") {
    return str;
  }
  return reverseStringRecursively(str.slice(1)) + str[0];
}

// Example usage:
console.log(reverseStringRecursively("hello")); // Output: "olleh"
```
This approach relies on slicing the string and appending characters in reverse order until the base case (empty string) is reached.

---

### **3. Using Reduce**
```javascript
function reverseStringUsingReduce(str) {
  return str.split('').reduce((reversed, char) => char + reversed, '');
}

// Example usage:
console.log(reverseStringUsingReduce("hello")); // Output: "olleh"
```
The `reduce` method accumulates characters in reverse order.

---

### **4. Using ES6 Spread Operator**
```javascript
function reverseStringWithSpread(str) {
  return [...str].reverse().join('');
}

// Example usage:
console.log(reverseStringWithSpread("hello")); // Output: "olleh"
```
The spread operator `...` creates an array from the string, which can then be reversed.

---

### **5. Using Stack**
```javascript
function reverseStringWithStack(str) {
  const stack = [];
  for (const char of str) {
    stack.push(char);
  }
  let reversed = '';
  while (stack.length > 0) {
    reversed += stack.pop();
  }
  return reversed;
}

// Example usage:
console.log(reverseStringWithStack("hello")); // Output: "olleh"
```
This uses a stack to reverse the string, emulating the LIFO (Last-In-First-Out) principle.
