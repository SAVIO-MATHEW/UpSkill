undefined vs not defined (ReferenceError)

undefined

Variable exists but has no value assigned

let a;
console.log(a); // undefined


Also happens when:

function test(x) {}
test(); // x === undefined


not defined

Variable does NOT exist in scope

Accessing it throws an error

console.log(b); // ReferenceError: b is not defined
