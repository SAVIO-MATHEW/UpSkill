Creating a pagination component allows users to navigate through data divided across multiple pages. Here's how you can implement a simple pagination component in JavaScript and React:

---

### **Example Code (React)**

```javascript
import React, { useState } from 'react';

const PaginationComponent = ({ data, itemsPerPage }) => {
  const [currentPage, setCurrentPage] = useState(1);

  // Calculate total pages
  const totalPages = Math.ceil(data.length / itemsPerPage);

  // Function to get current page data
  const getPageData = () => {
    const startIndex = (currentPage - 1) * itemsPerPage;
    const endIndex = startIndex + itemsPerPage;
    return data.slice(startIndex, endIndex);
  };

  // Function to handle page change
  const changePage = (page) => {
    if (page > 0 && page <= totalPages) {
      setCurrentPage(page);
    }
  };

  return (
    <div>
      {/* Display current page data */}
      <div>
        {getPageData().map((item, index) => (
          <div key={index}>{item}</div>
        ))}
      </div>

      {/* Pagination controls */}
      <div style={{ marginTop: "10px" }}>
        <button onClick={() => changePage(currentPage - 1)} disabled={currentPage === 1}>
          Previous
        </button>
        <span> Page {currentPage} of {totalPages} </span>
        <button onClick={() => changePage(currentPage + 1)} disabled={currentPage === totalPages}>
          Next
        </button>
      </div>
    </div>
  );
};

export default PaginationComponent;

// Example usage
const App = () => {
  const dummyData = Array.from({ length: 50 }, (_, i) => `Item ${i + 1}`);
  return <PaginationComponent data={dummyData} itemsPerPage={5} />;
};

export default App;
```

---

### **How This Works**
1. **Data Splitting:** The `getPageData()` function divides the full dataset into slices based on the current page number and the `itemsPerPage` value.
2. **Page Navigation:** `changePage()` updates the current page and ensures users don't navigate beyond the first or last page.
3. **Dynamic Rendering:** The component dynamically updates the visible data and controls based on user interaction.

---

### **Features You Can Add**
- **Pagination Numbers:** Display clickable page numbers for easy navigation:
  ```javascript
  {Array.from({ length: totalPages }, (_, i) => (
    <button key={i + 1} onClick={() => changePage(i + 1)}>
      {i + 1}
    </button>
  ))}
  ```
- **Styling:** Add CSS styles or libraries like Bootstrap for better design.
- **Server-Side Pagination:** If your dataset is large, fetch only the data for the current page from the server.
