Lifecycle methods are special functions or hooks provided in frameworks like React to manage the life stages of a component. These methods allow you to execute code at specific points during the component's lifespanâ€”such as initialization, updates, and cleanup.

### Lifecycle Phases in React:
1. **Mounting**: When a component is created and added to the DOM.
   - `constructor()`: Sets up initial state and binds methods (rarely used in modern React).
   - `static getDerivedStateFromProps()`: Updates state based on initial props.
   - `render()`: Builds and renders the UI.
   - `componentDidMount()`: Executes code after the component is added to the DOM (e.g., fetching data).

2. **Updating**: When a component's props or state changes.
   - `static getDerivedStateFromProps()`: Updates state when props change.
   - `shouldComponentUpdate()`: Determines if re-rendering is necessary (used for performance optimization).
   - `render()`: Re-renders the UI with updated data.
   - `getSnapshotBeforeUpdate()`: Captures information (e.g., scroll position) before the DOM updates.
   - `componentDidUpdate()`: Executes code after the DOM has been updated (e.g., handling side effects).

3. **Unmounting**: When a component is removed from the DOM.
   - `componentWillUnmount()`: Performs cleanup (e.g., stopping timers or removing event listeners).

### Example:
```javascript
import React, { Component } from "react";

class MyComponent extends Component {
  constructor(props) {
    super(props);
    this.state = { count: 0 };
    console.log("Constructor");
  }

  componentDidMount() {
    console.log("Component Mounted");
  }

  componentDidUpdate(prevProps, prevState) {
    console.log("Component Updated");
  }

  componentWillUnmount() {
    console.log("Component Will Unmount");
  }

  handleClick = () => {
    this.setState({ count: this.state.count + 1 });
  };

  render() {
    return (
      <div>
        <p>Count: {this.state.count}</p>
        <button onClick={this.handleClick}>Increase Count</button>
      </div>
    );
  }
}

export default MyComponent;
```

### Modern React with Hooks
In functional components, lifecycle methods are replaced by **Hooks**, such as:
- `useEffect`: Combines `componentDidMount`, `componentDidUpdate`, and `componentWillUnmount`.

Example:
```javascript
import React, { useState, useEffect } from "react";

function MyFunctionalComponent() {
  const [count, setCount] = useState(0);

  useEffect(() => {
    console.log("Effect: Component Mounted or Updated");

    return () => {
      console.log("Cleanup: Component Will Unmount");
    };
  }, [count]); // Dependency array

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>Increase Count</button>
    </div>
  );
}

export default MyFunctionalComponent;
```
