Insertion sort = build the sorted array one item at a time by inserting each element into its correct position.

ğŸ”¹ Idea (simple)

You split the array into:

Left side â†’ already sorted

Right side â†’ unsorted

Pick one element from the unsorted side and insert it into the correct place in the sorted side (like arranging playing cards in your hand).

ğŸ”¹ Steps

For each index i from 1 â†’ n-1:

Store current element â†’ key = arr[i]

Compare it with elements on the left

Shift larger elements one position right

Insert key in the gap

ğŸ”¹ Code (JS)
function insertionSort(arr) {
  for (let i = 1; i < arr.length; i++) {
    let key = arr[i];
    let j = i - 1;

    while (j >= 0 && arr[j] > key) {
      arr[j + 1] = arr[j]; // shift right
      j--;
    }

    arr[j + 1] = key; // insert
  }
  return arr;
}
