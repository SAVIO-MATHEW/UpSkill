The `for...in` and `for...of` loops in JavaScript are both used for iterating over items, but they have distinct purposes and behaviors. Here's how they differ:

---

### **`for...in`**
- **Purpose**: Used to iterate over the keys (properties) of an object.
- **Usage**: It works on objects and arrays, but when used on arrays, it iterates through the index (not the values).
- **Example**:
  ```javascript
  const obj = { a: 1, b: 2, c: 3 };

  for (let key in obj) {
    console.log(key); // Logs 'a', 'b', 'c'
    console.log(obj[key]); // Logs 1, 2, 3
  }

  const arr = [10, 20, 30];

  for (let index in arr) {
    console.log(index); // Logs 0, 1, 2 (indexes)
    console.log(arr[index]); // Logs 10, 20, 30
  }
  ```

---

### **`for...of`**
- **Purpose**: Used to iterate over the values of iterable objects (like arrays, strings, maps, sets, etc.).
- **Usage**: It directly accesses the values, making it ideal for working with arrays and other iterable data structures.
- **Example**:
  ```javascript
  const arr = [10, 20, 30];

  for (let value of arr) {
    console.log(value); // Logs 10, 20, 30
  }

  const str = "Hello";

  for (let char of str) {
    console.log(char); // Logs 'H', 'e', 'l', 'l', 'o'
  }
  ```

---

### **Key Differences**

| **Feature**             | **`for...in`**                     | **`for...of`**               |
|--------------------------|------------------------------------|------------------------------|
| Iterates Over            | Keys (indexes or property names). | Values (elements or items). |
| Operates On              | Objects and arrays.               | Iterable objects (arrays, strings, maps, sets, etc.). |
| Common Use               | Traversing object properties.     | Traversing array or iterable values. |

---

### **Summary**
- Use `for...in` for objects and when you need the keys.
- Use `for...of` for arrays, strings, or other iterables when you need the values.
