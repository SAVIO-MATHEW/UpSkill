## âœ… 1ï¸âƒ£ Return Value (Biggest Difference)

### `map()`

* Returns a **new array**
* Used when you want to **transform data**

```js
const nums = [1, 2, 3];

const doubled = nums.map(n => n * 2);
// [2, 4, 6]
```

---

### `forEach()`

* Returns **undefined**
* Used for **side effects only**

```js
const nums = [1, 2, 3];

nums.forEach(n => console.log(n));
```

---
